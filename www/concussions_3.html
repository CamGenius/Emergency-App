Concussions: Aftermath

Aftermath

What You Shouldn't Do

• Return to what you were doing while symptoms are still present

• Return to what you were doing on the same day of the injury

• Get another concussion especially while still suffering from symptoms for the last concussion


What you Should Do

• Seek an evaluation clearing to return to what you were doing

Chronic Symptoms

• Headaches

• Vertigo

• Depression

• Trouble problem solving

• Long-term memory loss

• There are many possible chronic symptoms as this is still an actively studied area
